# Visual TODO App

**AI生成画像で「着手の壁」を突破するタスク管理アプリ**

## 概要

「やるべきことは分かっているのに動けない」を解決。タスクの状況（いつ・どこで・何を）を入力すると、**画像生成AIが「作業中の自分」のイメージ画像を作成**。視覚化によって行動への心理的ハードルを下げます。

## 特徴

- **具体化**: When・Where・What を明確に定義
- **AI画像生成**: 入力した状況で作業している自分の画像を自動生成
- **集中**: 生成画像とともに今やるべきタスクを強調表示
